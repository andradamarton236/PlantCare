# PlantCare AI Frontend

Frontend React/Vite conectat la backend-ul FastAPI.

## Rulare

1. Pornește backend-ul în proiectul Python:

```bash
python -m uvicorn main_api:app --reload
```

2. Pornește frontend-ul:

```bash
npm install
npm run dev
```

Aplicația folosește API-ul de la `http://127.0.0.1:8000`.
